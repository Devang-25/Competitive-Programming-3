# Problem Description
Given a list of non negative integers 'A', arrange them such that they form the largest number.  

**Example**  
`Input: A = [17, 7, 2, 45, 72]`  
`Output: "77245217"`  

`Input: A = [10, 2]`  
`Output: "210"`  

**Leetcode Problem Link: https://leetcode.com/problems/largest-number/**
